# IPL Performance Analysis

A web-based IPL Performance Analysis platform developed using **Flask**, **HTML**, **CSS**, and **Tableau Public**. The application provides an interactive interface to explore IPL statistics and performance insights through a modern, responsive dashboard.

## Features

- Modern responsive landing page
- Interactive Tableau dashboard integration
- Flask-based web application
- Responsive dark-themed user interface
- Business Intelligence visualization

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Tableau Public
